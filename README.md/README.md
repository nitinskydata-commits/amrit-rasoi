# SBMI E-Commerce Platform - Full Stack MERN Application

## 🚀 Complete Production-Ready Amazon Clone

A fully functional e-commerce platform built with **MERN Stack** (MongoDB, Express, React, Node.js) featuring all modern e-commerce capabilities.

---

## ✨ Features

### Customer Features
- ✅ OTP-based login & Email/Password authentication
- ✅ Product browsing with advanced search & filters
- ✅ Product details with image gallery & reviews
- ✅ Shopping cart with quantity management
- ✅ Checkout with multiple payment options
- ✅ Order tracking with status updates
- ✅ User profile management
- ✅ Multiple delivery addresses with map integration
- ✅ Product reviews & ratings
- ✅ Responsive design for all devices

### Admin Features
- ✅ Product management (CRUD operations)
- ✅ Order management & status updates
- ✅ User management
- ✅ Review moderation

### Technical Features
- ✅ JWT authentication with secure cookies
- ✅ Protected routes & role-based access
- ✅ Redux Toolkit for state management
- ✅ MongoDB with Mongoose ODM
- ✅ RESTful API architecture
- ✅ Framer Motion animations
- ✅ React Toastify notifications
- ✅ Leaflet maps integration
- ✅ Image upload support (Cloudinary)
- ✅ Payment gateway integration ready (Razorpay)
